# 🌱 식물 CO2 분석 대시보드
# 당곡고 신재륜 & 수도여고 신승아
> Raspberry Pi Pico 2W + SCD30 센서를 이용한 실시간 CO2 모니터링 및 광합성 보조광 자동 제어 시스템

---

## 📋 프로젝트 개요

식물의 광합성 유형(CAM / C3 / C4)에 따라 시간대별로 예상되는 CO2 농도를 계산하고, 실제 측정값과 비교하여 광합성이 부족하면 NeoPixel LED를 자동으로 켜서 보조광을 공급하는 스마트 식물 관리 시스템입니다.

---

## 🔧 하드웨어 구성

| 부품 | 사양 | 연결 핀 |
|------|------|---------|
| 메인 보드 | Raspberry Pi Pico 2W | - |
| CO2 센서 | SCD30 (CO2 / 온도 / 습도) | I2C0 (GP8=SDA, GP9=SCL) |
| LED 스트립 | NeoPixel 60개 | GP16 |

배선도

Pico 2W → SCD30
GP8 (SDA) → SDA
GP9 (SCL) → SCL
3.3V → VCC
GND → GND

Pico 2W → NeoPixel
GP16 → DIN
5V → VCC
GND → GND

 ┣ 📄 main.py         ← 메인 실행 파일
 ┗ 📄 wifi_config.py  ← 와이파이 설정 파일

## ⚙️ 설치 및 설정

### 1단계 — wifi_config.py 작성

WIFI_SSID     = "와이파이이름"
WIFI_PASSWORD = "와이파이비밀번호"

### 2단계 — main.py 설정값 확인

CO2_BASE           = 800.0  (밀폐 용기 기준 CO2 ppm)
LIGHT_THRESHOLD    = 150.0  (보조광 ON 기준 차이값 ppm)
THRESHOLD_CAUTION  = 1000   (CO2 주의 기준 ppm)
THRESHOLD_DANGER   = 1500   (CO2 위험 기준 ppm)
SHEET_LOG_INTERVAL = 10     (구글 시트 전송 주기 초)

### 3단계 — Thonny에서 실행

1. wifi_config.py → Pico에 저장
2. main.py → Pico에 저장
3. main.py 실행 (▶ 버튼)

### 4단계 — 대시보드 접속

쉘에 뜨는 IP 주소를 브라우저에 입력
예) http://192.168.0.25

## 🌿 식물 유형별 동작

### CAM 식물 🌵 (선인장, 알로에)

| 시간대 | 기공 상태 | LED 동작 |
|--------|-----------|---------|
| 낮 (06~20시) | 닫힘 | 노랑 (정상) |
| 밤 (20~06시) | 열림 | 파랑 (흡수 중) |

CAM은 밤에 CO2를 흡수하므로 보조광 대신 상태 표시만 합니다.

### C3 식물 🌳 (나무, 상추)

| 상태 | 조건 | LED 동작 |
|------|------|---------|
| 광합성 정상 | 실제 CO2 <= 예상 + 150ppm | 초록~빨강 게이지 |
| 광합성 부족 | 실제 CO2 > 예상 + 150ppm | 밝은 흰색 (보조광) |

### C4 식물 🌾 (옥수수, 사탕수수)

C3와 동일한 방식으로 동작합니다. (광포화점이 더 높음)

## 💡 보조광 시스템 원리

현재 시각 확인 (NTP 동기화)
        ↓
시간대별 예상 CO2 계산
예상 CO2 = CO2_BASE(800) + 시간대별 오프셋
        ↓
실제 CO2 측정값과 비교
        ↓
차이 > 150ppm ?
  YES → 광합성 부족 → LED 흰색 점등 (보조광 공급)
  NO  → 정상       → LED 게이지 표시

### 시간대별 예상 CO2 오프셋 (C3 기준)

| 시간 | 오프셋 | 의미 |
|------|--------|------|
| 0~5시 | +50~60ppm | 야간 호흡, CO2 증가 |
| 6~11시 | -20~100ppm | 광합성 활발, CO2 감소 |
| 12~17시 | -80~90ppm | 광합성 지속 |
| 18~23시 | +10~50ppm | 저녁, 호흡 시작 |

## 🔴 LED 색상 의미

| 색상 | 의미 |
|------|------|
| 밝은 흰색 | 보조광 공급 중 (광합성 부족) |
| 초록~빨강 게이지 | 광합성 정상 (CO2 농도 표시) |
| 파랑 | CAM 야간 흡수 중 |
| 노랑 | CAM 주간 (기공 닫힘) |
| 노랑 깜빡임 | CO2 주의 (1000ppm 이상) |
| 빨강 빠른 깜빡임 | CO2 위험 (1500ppm 이상) |

## 📊 대시보드 기능

|------|------|

## 📡 구글 시트 연동 전송 데이터

| 파라미터 | 설명 |
|---------|------|
| co2 | CO2 측정값 (ppm) |
| temp | 온도 (°C) |
| humi | 습도 (%) |
| plant | 식물 유형 (CAM / C3 / C4) |
| expected | 예상 CO2 (ppm) |
| diff | 실제 - 예상 차이 (ppm) |
| light | 보조광 상태 (ON / OFF) |

## ⚠️ 주의사항

- Wi-Fi는 2.4GHz 대역만 연결 가능 (5GHz 불가)
- NTP 시간 동기화 실패 시 수동으로 RTC 시각 설정 필요
- 밀폐 용기 실험 시 CO2가 누적될 수 있음
- SCD30 센서는 연결 후 약 10초 워밍업 필요
